Agent 系统工程师（深圳）
岗位职责
1、主导智能 Agent 系统的集成与交付全流程，严格遵循标准化工程规范，完成机器人动作执行、环境导航、语音交互等核心能力模块的系统化整合，搭建可自主解析用户意图、规划并执行任务的智能 Agent 解决方案，保障系统在多业务场景下的稳定可靠运行。
2、负责大模型（LLM/VLA 等）在机器人端的全链路落地，涵盖模型推理部署、定制化微调、性能基准测试（Benchmark）等核心工作，深度分析模型运行过程中的性能瓶颈，输出针对性优化方案，持续提升大模型在机器人场景的应用效果。
任职要求
1、本科及以上学历，计算机科学与技术、软件工程、人工智能等相关专业背景，具备扎实的计算机底层基础与工程化思维；
2、精通 LLM、VLA 等大模型的推理与部署全流程，熟练掌握 vllm、sglang 等主流推理框架的使用与调优技巧；
3、深入理解 LLM 工具调用（Function Calling）核心原理，具备相关功能的工程化实现与落地经验；
4、熟悉智能 Agent 架构设计逻辑，掌握 Agent 系统工程化落地的方法与实践经验；
5、具备扎实的多模块、多系统对接联调能力，能高效解决集成过程中的各类技术问题；
6、具备强烈的业务感知能力与交付导向思维，执行力强、注重细节，拥有优秀的编程能力与问题排查能力。

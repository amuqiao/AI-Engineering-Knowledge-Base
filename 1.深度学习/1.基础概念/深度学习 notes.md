**③ 机制深挖**（按实际模型家族展开，未涉及的跳过）

- **若是 Transformer / 注意力机制** → 分析注意力变体（MHA/MQA/GQA）、
  位置编码方案（RoPE/ALiBi/绝对位置）、归一化策略、KV Cache 设计
- **若是扩散模型** → 分析正向加噪过程、噪声调度策略（线性/余弦）、
  条件控制机制（CFG/ControlNet）、采样算法（DDPM/DDIM/DPM-Solver）
- **若是 CNN / 检测网络** → 分析感受野演进逻辑、特征金字塔结构、
  归纳偏置设计，以及检测头和后处理（NMS）
- **若是 MoE（混合专家）** → 分析路由机制（Top-K 门控）、
  专家负载均衡策略、稀疏激活的训练稳定性问题
- **若是多模态模型** → 分析模态对齐机制、跨模态融合策略、
  对比学习目标（CLIP式）或生成式对齐
- **若是 SSM / 线性注意力** → 分析状态空间方程、并行扫描实现、
  与 Attention 的近似关系
- **若是自回归生成模型** → 分析解码策略（贪心/束搜索/温度采样/Top-p），
  推理时的 KV Cache 机制，以及长文本外推能力
- **若是 RNN / LSTM / GRU / seq2seq** → 分析门控机制（输入门/遗忘门/输出门）、
  梯度消失与长程依赖问题、双向结构设计、
  Encoder-Decoder 对齐机制（Bahdanau / Luong Attention）
- **若是语音模型** → 分析特征提取方式（MFCC / Mel 频谱 / 原始波形 CNN），
  帧级建模 vs 段级建模，CTC / AED / RNN-T 等解码目标，
  预训练自监督目标，流式推理与延迟控制
- **若是时序模型** → 分析序列建模策略（局部卷积 / 全局注意力 / 状态空间），
  时间步粒度与 Patch 化设计，单变量 vs 多变量建模，
  预测头设计（直接多步输出 vs 迭代递推），分布偏移应对策略
